Reward guarantee

I provide a guarantee for any staking rewards that are lost due to server downtime. As such it is no longer necessary to micromanage your pool selection.

An automatic rebate is provided for these lost rewards, paid directly to the wallet address associated with your stake key, once an epoch. If multiple addresses are associated with the staking key, the latest address is used. A record of these rebates can be obtained by contacting me.
